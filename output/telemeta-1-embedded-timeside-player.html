Since the release of Telemeta 1.0, it is possible to embed a TimeSide UI dynamical player in any HTML page :)

Like this one from the CREM ethno-musicology archives, GAMELAN PEGULINGAN Bali 1957:

All metadata time markers can be displayed during playback so you can follow, jump, read as you are listening to the sound. Here is a nice example with "I Could Never Make that Music Again" from L&R and RadioMentale (Sub Rosa SR 249, 2007). This track is a radio collage which organises and disorganises quotes from various famous electronic music artists interviewed and recorded:
